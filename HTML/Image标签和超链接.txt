图片标签
	img    
		src : 图片路径
		本地资源路径：一般本地图片资源使用相对路径即可
		网络资源路径：图片资源的URL地址
	属性：
     		<img    src=""   width= ""   height="">
		width : 设置图片的宽度，如果是单独设置，则在图片保证不失真的状态下自动缩小或者扩大
			单位可以的 px (占屏幕)或者是百分比（占浏览器）
		height : 设置图片的高度，如果是单独设置，则在图片保证不失真的状态下自动缩小或者扩大
			单位可以的 px (占屏幕)或者是百分比（占浏览器）
		title : 图片的标题 （当鼠标点击图片时会显示图片的标题）
		alt :  当图片不存在的情况下 ， 给出自我定义的内容进行提示

	注意：
		相对路径：从当前文件出发找另一个文件所在的位置
		绝对路径：从根盘符出发找文件的路径
		图片是不会自动换行的（行内元素）



超链接标签：
	a标签 
		href：要跳转的网页资源路径
			本地资源：相对路径
			网络资源：网络资源的URL
		target : 指定要跳转的网页资源的显示位置
			_blank : 新标签页中显示出跳转的界面
			_self : 在当前页中刷新显示
			_top : 在顶层页面中显示
			_parent : 在父界面中显示
		
	锚点学习：
		作用：在一张网页中进行资源跳转
		使用：
			先使用超链接标签在指定的网页位置，添加name属性，然后将主的href中键入name属性中的值
		 例如;  <a href = "#锚点名"> </a>  //此时点击此处可以瞄点到第一章的内容 ，注意一定要加  #  号
		          <p> 第一章</p>
		          <a  name="锚点名"> </a>   //此位置可以放在段落前，也可以跳到段落后  
	回到顶部 ：# 相当于网页自己跳自己
		格式：
			<a href="#">回到顶部 </a>		
			
	<a href=""  > 点击此处内容进行超链接 (必须写，不然用户无法点击进行链接跳转) </a>
		注意：
			超链接中一定要什么申明访问方式，访问方式可以是文字也可以是图片
